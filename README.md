# 🌸 Iris Flower Classification

## 📌 Project Overview
This project is part of the CodeAlpha Data Science Internship.  
The goal is to build a machine learning model to classify iris flowers into three species based on their physical features.

---

## 🎯 Objective
To classify iris flowers into:
- Iris-setosa
- Iris-versicolor
- Iris-virginica  

using machine learning techniques.

---

## 📊 Dataset
The dataset contains the following features:
- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width  

Target variable:
- Species of the flower  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Jupyter Notebook  

---

## 🧠 Model Used
- K-Nearest Neighbors (KNN)

---

## ⚙️ Steps Performed
1. Loaded the dataset (Iris.csv)  
2. Cleaned the data  
3. Split into training and testing sets  
4. Trained KNN model  
5. Evaluated model performance  

---

## 📈 Results
- Achieved **100% accuracy** on test data  
- Model successfully classified all three species  

---

## 🔍 Insights
- Iris dataset is well-structured and easy to classify  
- Petal features play a key role in classification  

---

## 🚀 Conclusion
The model performed very well and demonstrated how machine learning can be used for classification problems.
